Understanding These Metrics

These metrics measure the difference between the AI-predicted score and the human-graded score on the ASAP dataset. Lower values on both MAE and RMSE indicate a more precise and reliable model.

MAE
Average absolute difference between predicted and actual scores.
RMSE
Penalises larger errors more heavily than MAE; sensitive to outliers.
Goal
Both values should be as close to 0 as possible for best accuracy.
